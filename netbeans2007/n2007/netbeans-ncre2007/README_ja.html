NetBeans[tm] IDE 5.0

NetBeans IDE は、Java[tm] プラットフォーム用のデスクトップアプリケーションおよび Web アプリケーションを作成、コンパイル、テスト、デバッグするための統合開発環境 (IDE) です。NetBeans IDE には、構文強調表示やエラーチェックなどの十分な機能を備えたテキストエディタや、ビジュアルデザインツール、Ant サポート、バージョン管理システムサポートなどの多くの機能があります。

Microsoft Windows の場合、IDE は次のどの方法でも起動できます。

  • デスクトップ上の NetBeans 5.0 のアイコンをダブルクリックする。
  • 「スタート」>「プログラム」>「NetBeans 5.0」>「NetBeans IDE」を選択する。
  • コマンド行から起動する (C:\> netbeans-install-directory\bin\netbeans.exe)。

Solaris OS または Linux の場合は、次の方法で IDE を起動できます。

  • コマンドプロンプトで /netbeans-install-directory/bin/netbeans と入力する。

IDE を終了する方法は、次のとおりです。

  • IDE から「ファイル」>「終了」を選択する。

Microsoft Windows の場合、IDE をアンインストールする方法は次のとおりです。

  • 「アプリケーションの追加と削除」ユーティリティーを使用する。ディレクトリやファイルを手動で削除しないでください。

Solaris OS または Linux の場合、IDE をアンインストールする方法は次のとおりです。

/netbeans-install-directory/_uninst にある付属のアンインストーラを使用する。ディレクトリやファイルを手動で削除しないでください。

NetBeans IDE に関するマニュアル等の入手場所

  • NetBeans IDE 5.0 のリリース情報は、http://www.netbeans.org/community/releases/50/index.html を参照してください。
  • IDE における基本的な作業の流れを扱った簡単な手引き書が http://www.netbeans.org/kb/50/quickstart_ja.html にあります。このマニュアルは、ほかのマニュアルへのリンクも含まれています。
  • IDE には、ヘルプが用意されています。メインメニューから「ヘルプ」>「ヘルプの目次」を選択してください。「ヘルプ」メニューにも、ほかの有用なドキュメントへのリンクが含まれています。
