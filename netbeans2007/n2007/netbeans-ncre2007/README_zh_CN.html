NetBeansTM IDE 5.0

NetBeans IDE 是一个集成开发环境 (Integrated Development Environment, IDE)，用于编写、编译、测试和调试 JavaTM 平台的桌面应用程序和 Web 应用程序。NetBeans IDE 包含一个功能完善的文本编辑器，其中包括语法突出显示和错误检查、可视化设计工具、Ant 支持、版本控制系统支持以及许多其他功能。

使用以下某个方法启动 IDE (Microsoft Windows)：

  • 双击桌面上的 NetBeans 5.0 图标。
  • 选择“开始”>“所有程序”> "NetBeans 5.0" > "NetBeans IDE"。
  • 通过命令行 C:\> <netbeans 安装目录>\bin\netbeans.exe 启动 IDE。

启动 IDE（Solaris OS 或 Linux）：

  • 在命令提示符下键入： /<netbeans 安装目录>/bin/netbeans

停止 IDE：

  • 从 IDE 中选择“文件”>“退出”。

卸载 IDE (Microsoft Windows)

  • 使用“添加/删除程序”实用程序。不要手动删除目录和文件。

卸载 IDE（Solaris OS 或 Linux）

使用 /<netbeans 安装目录>/_uninst 中提供的卸载程序。不要手动删除目录和文件。

查找 NetBeans IDE 文档

  • 可以在以下位置找到有关 NetBeans IDE 5.0 的发行信息 http://www.netbeans.org/community/releases/50/index.html。
  • http://www.netbeans.org/kb/50/quickstart_zh_CN.html 中提供了介绍 IDE 中基本工作流的简短指南。此文档还提供了指向其他文档的链接。
  • IDE 中提供了帮助文档。从主菜单中，选择“帮助”>“帮助内容”。“帮助”菜单还提供了指向其他相关文档的链接。
